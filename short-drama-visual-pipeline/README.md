# 🎬 短剧视觉流水线 (Short Drama Visual Pipeline)

面向 AI 短剧承制团队的生产力工具 Skill，将文学剧本自动转化为结构化分镜脚本，输出高精度文生图 Prompt，并通过全局角色特征库确保跨场次角色视觉一致性。

## ✨ 核心功能

| 模块 | 说明 |
|------|------|
| **剧本结构化分析** | 自动拆解场次、角色、动作、情绪、关键道具 |
| **角色三视图 Prompt** | 正面/侧面/背面独立 prompt + 三视图总汇 prompt（一张图出三角度） |
| **场景四视图 Prompt** | 正面全景/侧面/俯瞰/反打独立 prompt + 2×2 网格总汇 prompt |
| **角色表演分镜 Prompt** | 按剧情节奏生成带镜头语言的表演特写 prompt |
| **全局一致性控制** | 角色特征卡 + 一致性追踪表，确保跨场次角色视觉稳定 |

## 🎨 支持风格

- 电影写实（默认）
- 国风工笔
- 赛博朋克
- 日系动漫
- 暗黑哥特
- 清新治愈

也支持自定义风格关键词。

## 📦 安装

本技能为 [Coze](https://www.coze.cn) 平台 Skill。

1. 进入 Coze 控制台 → 我的技能 → 导入技能
2. 选择本项目中的 `SKILL.md` 文件导入
3. 导入后即可在对话中使用

## 🚀 使用方式

在 Coze 对话中直接发送剧本内容或上传剧本文件（支持 `.docx`、`.txt`），技能会自动启动以下流水线：

```
剧本上传 → 风格确认 → 剧本结构化分析 → 角色特征库建立
→ 角色三视图 Prompt → 场景四视图 Prompt → 角色表演分镜 Prompt
→ 一致性追踪表 → 输出完整分镜脚本包
```

### Prompt 输出格式

**角色三视图（每位角色 4 条）：**
- 正面视图 prompt
- 侧面视图 prompt
- 背面视图 prompt
- **三视图总汇 prompt**（一张图横向排列三视角，turnaround sheet）

**场景四视图（每个场景 5 条）：**
- 正面全景 prompt
- 侧面视角 prompt
- 俯瞰全景 prompt
- 反打视角 prompt
- **四视图总汇 prompt**（2×2 网格一张图出四个角度）

**角色表演分镜（每个关键表演 1 条）：**
- 含镜头语言、情绪描述、角色特征引用、Midjourney 参数

### 输出产物

最终输出包含以下部分：

1. **角色特征库** — 全维度角色视觉描述
2. **角色三视图 Prompt 包** — 独立视角 + 总汇
3. **场景四视图 Prompt 包** — 独立视角 + 总汇
4. **角色表演分镜 Prompt** — 含一致性控制参数
5. **角色一致性追踪表** — 跨场次特征锚定清单

## ⚙️ 技术细节

- 默认使用 **电影写实** 风格（cinematic, photorealistic, 8K）
- Midjourney 参数规范：
  - 场景全景：`--ar 16:9 --s 250 --q 2`
  - 角色特写：`--ar 2:3 --s 200 --q 2`
  - 三视图总汇：`--ar 16:9 --s 200 --q 2`
  - 四视图总汇：`--ar 16:9 --s 250 --q 2`
- 统一 Negative Prompt 控制画面质量
- 角色一致性通过特征描述词锚定 + `--cref` 参数建议

## 📋 系统要求

- Coze 平台账号
- 无需外部 API（使用 Coze 内置 LLM 能力）
- 图像生成需用户自行使用 Midjourney / SD / FLUX 等工具
